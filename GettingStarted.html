# Getting Started with Vue Styleguidist

# 1. Install Styleguidist

Install webpack if you don’t have it already:

npm install --save-dev webpack

Install Styleguidist:

npm install --save-dev vue-styleguidist

# 2. Configure your style guide

If you’re using Vue-CLI 3.x you can skip this step, and install vue-cli-plugin-styleguidist.

Point Styleguidist to your Vue components and tell how to load your code.

# 3. Add npm scripts for convenience

Add these scripts to your package.json:

{
  "scripts": {
+    "styleguide": "vue-styleguidist server",
+    "styleguide:build": "vue-styleguidist build"
  }
}

# 4. Start your style guide

Run npm run styleguide to start style a guide dev server.

Run npm run styleguide:build to build a static version.
